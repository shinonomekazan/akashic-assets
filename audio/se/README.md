## 参考

m4aを先に用意し、libvorbisがある状態のffmpegをインストールした環境のWSL2等で、以下のように実行すれば、oggファイルを一括用意可能。

```sh
ls m4a/*.m4a | xargs -i basename {} .m4a | xargs -i ffmpeg -i m4a/{}.m4a -acodec libvorbis ogg/{}.ogg
```
